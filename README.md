# fabric-peer-lib
Set of libraries that extends the core peer capabilities.

```
git clone https://github.com/trustbloc/fabric-peer-ext.git

cd fabric-peer-ext

# run linters
make checks

# run unit-test
make unit-test
```

## Build dependencies

* Go `1.11.5`
* Docker `18.09.x` or above
